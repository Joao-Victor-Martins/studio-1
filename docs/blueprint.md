# **App Name**: DiameterCalc Pro

## Core Features:

- Diameter Selection: Allows users to select minimum and maximum diameter options from predefined choices.
- Diameter Input: Numeric input field for users to enter the current diameter value in millimeters.
- Weight Input: Numeric input field for users to enter the total weight in kilograms.
- Diameter Percentage Calculation: Calculates the percentage of the current diameter within the selected minimum and maximum diameter range.
- Weight Balance Calculation: Calculates the weight balance proportional to the calculated diameter percentage.
- Result Display: Displays the selected minimum diameter, selected maximum diameter, entered current diameter, calculated diameter percentage (2 decimal places), entered total weight, and calculated weight balance (2 decimal places) in a clear card format.
- Input Validation: Prevents users from entering a current diameter value outside the selected minimum and maximum diameter range.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey reliability and precision.
- Background color: Very light blue (#E8EAF6) to provide a clean and calm backdrop.
- Accent color: Bright orange (#FF9800) for the 'Calculate' button to draw attention.
- Body and headline font: 'Inter' for a modern, neutral and readable design.
- Simple and responsive layout with a card to display results below the input form.
- Use clear, functional icons to represent input fields and results.
- Subtle transitions when displaying results after calculation.
